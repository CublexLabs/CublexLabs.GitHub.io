How to make a Dialog

Creating a dialog with CubDialogs is easy! Follow these steps: 1. Define your dialog in a YAML file. 2. Specify the dialog type (Notice, Confirmation, or Multi Action). 3. Customize the dialog content, including text, buttons, and actions. 4. Use the API to trigger the dialog in your Minecraft server.
