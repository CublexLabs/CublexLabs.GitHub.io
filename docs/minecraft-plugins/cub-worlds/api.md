## API

Add this to your build.gradle:

```java

    maven("")
}

dependencies {
    compileOnly("io.github.devbd1:cubWorlds:<version>")
}
```

The latest version available on the repo can be found [here](https://github.com/CublexLabs/CubWorlds/tags)
